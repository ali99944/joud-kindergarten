# خطوات طفلي – Kindergarten Center Website Guide
## Arabic RTL Landing Page (Next.js + TailwindCSS)

---

## 1. Project Overview

**Project Name:** مركز خطوات طفلي  
**Arabic Name (Primary):** خطوات طفلي  
**Type:** Kindergarten / Early Childhood Education Center  
**Website Type:** Arabic Landing Page  
**Direction:** RTL (Right To Left)  
**Target Audience:** Parents of children (ages 3–6)

The website should feel:
- Friendly
- Playful
- Safe
- Educational
- Modern but soft (not corporate)

---

## 2. Visual Identity Reference

The UI design must **closely follow** the provided reference image in:
- Color palette
- Card styles
- Rounded shapes
- Illustration usage
- Soft gradients
- Child-friendly spacing

### Design Keywords
- Rounded
- Soft shadows
- Pastel colors
- Playful icons
- Floating elements
- Friendly typography

---

## 3. Colors (Based on Reference Image)

Use a **soft pastel palette** similar to the reference:

### Primary Colors
- Primary Purple: `#5B3FBF`
- Soft Purple Background: `#F4F1FF`
- Accent Orange: `#FF8A3D`
- Accent Yellow: `#FFC83D`

### Supporting Colors
- Dark Text: `#1F1F2E`
- Muted Text: `#6B6B85`
- Card Background: `#FFFFFF`
- Light Border: `#E8E6F3`

> Avoid harsh colors. Everything should feel soft and friendly.

---

## 4. Typography (Arabic Font Choice)

### Primary Arabic Font (Recommended)
**Cairo**
- Friendly
- Modern
- Highly readable
- Perfect for educational & kids platforms

Alternative (optional):
- **Tajawal** (softer)
- **Almarai** (rounded & playful)

### Font Usage
- Headings: Cairo Bold
- Subheadings: Cairo SemiBold
- Body text: Cairo Regular
- Buttons: Cairo Medium

---

## 5. Tech Stack Rules

- Framework: **Next.js (App Router)**
- Styling: **Tailwind CSS only**
- ❌ No shadcn
- ❌ No UI libraries
- ❌ No Material UI
- RTL enabled globally
- Mobile-first responsive design

---

## 6. Layout Direction & Structure

- Entire website must be **RTL**
- Text aligned to the right
- Icons should respect RTL direction
- Layout mirrors the reference design but adapted to Arabic flow

---

## 7. Page Structure (Landing Page Sections)

### 1. Navbar
- Logo (provided logo)
- Links (Arabic):
  - الرئيسية
  - من نحن
  - البرامج
  - الأنشطة
  - تواصل معنا
- CTA Button: **سجل طفلك الآن**

---

### 2. Hero Section

**Main Headline (Arabic):**
> مستقبل أفضل لطفلك

**Subheadline:**
> نؤمن أن كل خطوة في طفولة طفلك تصنع مستقبله

**CTA Buttons:**
- ابدأ الآن
- تواصل معنا

**Visual Style:**
- Soft background
- Rounded illustration areas

Use rounded cards with icons:

- +100 طفل سعيد
- +10 برامج تعليمية
- بيئة آمنة 100%
- فريق متخصص

Cards should float slightly with soft shadows.

### 4. Programs Section

- تنمية التفكير الإبداعي
- المهارات الاجتماعية
- الاستعداد المدرسي

Each card:
- Rounded
- Icon at top
- Hover animation (lift + glow)
- Soft gradient background

### 5. Activities / Resources Section

### 8. Footer

- Logo

---

## 8. UI Components Style Guide

### Buttons
- Fully rounded
- Gradient or solid purple
- White text
- Soft hover animation

### Cards
- Border-radius: large
- Shadow: very soft
- Hover: translateY(-4px)

### Icons
- Outline or soft-filled
- Rounded background circle
- Pastel colors

---

## 9. Animation & Interaction

- Subtle hover effects
- Floating decorative elements
- Smooth transitions
- No heavy motion

---

## 10. Accessibility & UX

- Large readable text
- Clear CTAs
- Simple language
- High contrast for text
- Touch-friendly buttons

---

## 11. Content Tone (Arabic)

- Warm
- Encouraging
- Parent-friendly
- Simple Arabic (Modern Standard Arabic)
- Avoid complex terms

---

## 12. Final Notes

- Follow the reference image visually
- Keep everything playful but clean
- Arabic-first design (not translated UI)
- Mobile experience is priority

---

**Goal:**  
Create a lovable, trustworthy kindergarten website that makes parents feel safe and excited about enrolling their child at **مركز خطوات طفلي**.
